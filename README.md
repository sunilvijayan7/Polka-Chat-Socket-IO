# Polka-Chat-Socket-IO
A Simple chatroom application using Polka (Light Weight Server) and Socket.io.

<p>
<h4>Setup</h4>
</p>
<p>
$ npm install
$ npm start
</p>

<p>
<h4>Usage</h4>
</p>
<p>
Open (multiple) browsers to localhost:3000!
</p>
